## About

This session provides a quick refresher on machine learning, highlighting their types, core principles and differences. By the end of this session, you will be ready to **explore advanced AI topics** by building a foundational understanding of machine learning that is required for subsequent modules. You will also be able to participate in technical discussions around machine learning and communicate your opinions and suggestions with conceptual clarity by:
- **Recalling supervised learning concepts** by emphasizing key characteristics and applications in classification and regression tasks.
- **Explaining unsupervised learning techniques** by focusing on clustering and dimensionality reduction methods.
- **Analyzing reinforcement learning frameworks** by emphasizing policy-based and reward-driven decision-making systems.
- **Applying ML concepts** to real-world use cases to contextualize learning.
